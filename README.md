# CrossBarWarrier

## ゲーム概要
CrossBarWarrierは、2人対戦の戦略的パズルゲームです。赤プレイヤー（先手）と緑プレイヤー（後手）が交互に線を引き、自分の色の点を繋げて勝利を目指します。

## ゲームの目的
- **赤プレイヤー（先手・☗）**: 盤面の上端から下端まで赤い点を縦に繋ぐ
- **緑プレイヤー（後手・☖）**: 盤面の左端から右端まで緑の点を横に繋ぐ

## プレイ方法

### ゲームの開始
1. ウィンドウ上部のスピナー（数値入力欄）でレベル（1～10）を選択
   - レベルによって盤面のサイズが変わります（レベル × 2 + 1）
   - レベル1: 3×3、レベル2: 5×5、レベル3: 7×7...
2. **開始**ボタンをクリックしてゲームスタート

### 盤面の説明
- **赤い点**: 先手プレイヤーが使用する点（偶数行・奇数列に配置）
- **緑の点**: 後手プレイヤーが使用する点（奇数行・偶数列に配置）
- 点と点の間の空きマスに線を引いていきます

### 手番のルール
現在の手番はウィンドウ上部に表示されます（☗番 or ☖番）

#### 先手（赤・☗）の場合
- 空いているマスをクリックして線を引きます
- **縦線**: 上下に赤い点があるマスに引けます
- **横線**: 左右に赤い点があるマスに引けます

#### 後手（緑・☖）の場合
- 空いているマスをクリックして線を引きます
- **縦線**: 上下に緑の点があるマスに引けます
- **横線**: 左右に緑の点があるマスに引けます

### 勝利条件
線を引いた直後に自分の勝利条件を満たすと勝ちです：
- **赤プレイヤー**: 盤面の最上行の赤い点から最下行の赤い点まで、赤い点と線で繋がったパスができた場合
- **緑プレイヤー**: 盤面の最左列の緑の点から最右列の緑の点まで、緑い点と線で繋がったパスができた場合

### ゲームの終了
- どちらかのプレイヤーが勝利条件を満たすと、ステータス表示に「☗勝ち」または「☖勝ち」と表示されます
- 新しいゲームを始めるには、レベルを選択して**開始**ボタンをクリックしてください

## 戦略のヒント
- 自分の道を作ると同時に、相手の道を遮断することが重要です
- 線は引いた後には削除できないので、慎重に考えて配置しましょう
- より大きなレベルでプレイすると、より複雑な戦略が必要になります

## 技術情報
- **開発環境**: Visual Studio (C++)
- **フレームワーク**: Win32 API
- **対応OS**: Windows
